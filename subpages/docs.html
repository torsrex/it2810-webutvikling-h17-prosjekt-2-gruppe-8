  • Bruker Node.js med Express som webserver:

    
    const express = require('express')
    const path = require('path')
    const opn = require('opn')
    const app = express()

    /* serves main page */
    app.get("/", (req, res) => {
        res.sendFile(__dirname + '/index.html')
    })

    /* serves all the static files */
    app.get(/^(.+)$/, (req, res) => {
        console.log('Request file ' + req.path)
        res.sendFile(__dirname + req.params[0])
    })

    app.listen(8082, () => {
        console.log('Web server listening on port 8082!\n')
    })

    opn('http://localhost:8082').then(() => {
        console.log("Opening http://localhost:8082 in browser...\n")
    })

  • Vi har brukt CSS Flexbox og Mobile First approach slik at siden tilpasser seg bra på alle enheter. Med Mobile First approach menes at siden tilpasses først til mobile enheter, også skaleres den opp vha. Media Queries.

    Her er et eksempel (hamburger menyen vises kun på mobile enheter):

    
    @media (min-width: 768px) {
      #hamburger {
        display: none
      }
    }

  • Navigasjonsbaren er implementert fra bunnen av. jQuery sin event listener ble brukt til å laste inn sider dynamisk med AJAX. Kodesnutten under viser også eksempel på bruk av jQuery selector-funksjonaliteten og bruk av funksjon som parameter. 

    
    $("nav").find("li a").on("click", e => { //Funksjon som parameter til on()
        e.preventDefault();
        const subPage = e.target.getAttribute("href")
        $("#mainContent").load(`subpages/${subPage}.html`, () => {
          switch (subPage) {
            case "docs":
              $.getScript("js/prism.js");
              break;
            case "media":
              $.getScript("js/media.js");
              break;
            default:
              return;
          }
        })
      })
